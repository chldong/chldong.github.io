关于博客系统的搭建

## 关键步骤

### 本地安装Jekyll

```bash
~ $ gem install jekyll
~ $ jekyll new myblog
~ $ cd myblog
~/myblog $ jekyll serve
# => Now browse to http://localhost:4000
```
